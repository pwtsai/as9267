About ISIF Asia SDX Project (AS9267)

  • This project aims to build an educational Internet eXchange playground and assign IPv6 resources to participants.
  • Providing online training courses with lab tutorials (i.e., SDN, BGP, RPKI/ROV, and Advanced SDN).
  • The project is supported by Information Society Innovation Fund (ISIF Asia) program of APNIC Foundation.

Facility

  • Locations:
    • Malaysia (Selangor) - MYREN
    • Taiwan (Tainan) - TWAREN
    • United States (Fremont) - HE/FMT2
